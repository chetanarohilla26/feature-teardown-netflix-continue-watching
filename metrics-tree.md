### Metric Ownership

| Metric Category | Purpose |
|----------------|---------|
| North Star Metric | Measures long-term user value delivered by Netflix |
| Primary Experiment Metric | Determines whether the Continue Watching redesign improves viewing continuity |
| Engagement Metrics | Measures depth of content consumption |
| Retention Metrics | Measures return behavior and platform loyalty |
| Guardrail Metrics | Ensures the feature does not hurt content discovery or user experience |
| Business Metrics | Measures impact on subscriber retention and revenue |
