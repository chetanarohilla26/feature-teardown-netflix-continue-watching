# Netflix Continue Watching Metrics Tree

## North Star Metric

### Viewing Hours Per User

```text
Viewing Hours Per User
│
├── Resume Starts
│   ├── Continue Watching CTR
│   ├── Resume Completion Rate
│   └── Time to Resume
│
├── Session Length
│   ├── Average Watch Time
│   ├── Episodes Per Session
│   └── Viewing Minutes
│
├── Retention
│   ├── D7 Retention
│   ├── D30 Retention
│   └── Monthly Active Users
│
└── Engagement
    ├── Weekly Active Users
    ├── Return Frequency
    └── Sessions Per User
```

## Input Metrics

Metrics the product team can influence directly:

- Continue Watching CTR
- Resume Starts
- Time to Resume
- Homepage Visibility

## Output Metrics

Business outcomes generated by the feature:

- Viewing Hours Per User
- Retention
- Session Length
- Subscription Stickiness

## Guardrail Metrics

- Discovery Rate
- Recommendation Diversity
- User Satisfaction
