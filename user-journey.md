# User Journey Analysis

## User Goal

Resume content quickly after an interruption.

---

## Current User Journey

```text
User Starts Show
        ↓
Leaves Session
        ↓
Time Gap
        ↓
Returns To Netflix
        ↓
Continue Watching Row Displayed
        ↓
One Click Resume
        ↓
Continued Viewing
        ↓
Higher Watch Time
```

---

## Friction Points

### Before Feature

```text
Return To Netflix
        ↓
Search Content
        ↓
Find Correct Episode
        ↓
Resume Watching
```

Problems:

- Time-consuming
- High effort
- Increased abandonment risk

---

## Benefits After Feature

### User

- Reduced effort
- Faster access
- Better experience

### Business

- Increased engagement
- Higher retention
- More viewing hours
